Lesson 2: HTML document structure

  • Every document will start with the >HTML< tag, Which incloses all of your html code
  • Next comes the >head< tag. This is where you would include 'information' about your page like a title (tutorial page). Another thing that goes into the header tag is "charset" which is the most commonly used.
  • The >body< tag is used to hold all the 'content' of your web page
  • Lastly is the >!DOCTYPE< tag which is always located at the top of your document and tell your documents that it is ok to use the lastest version of HTML
